# 📱 ICONOS PWA - KAPO RECLUTAMIENTO

## 📋 ICONOS REQUERIDOS

Los siguientes archivos deben ser reemplazados con iconos PNG reales:

### **Iconos Obligatorios**
- `icon-192x192.png` - 192x192px (PWA estándar)
- `icon-512x512.png` - 512x512px (PWA estándar)
- `apple-touch-icon.png` - 180x180px (iOS)

### **Especificaciones Técnicas**

#### **icon-192x192.png**
- **Tamaño**: 192x192 píxeles
- **Formato**: PNG con transparencia
- **Uso**: Icono principal de la PWA
- **Recomendación**: Logo de la aplicación centrado

#### **icon-512x512.png**
- **Tamaño**: 512x512 píxeles
- **Formato**: PNG con transparencia
- **Uso**: Icono de alta resolución para displays retina
- **Recomendación**: Logo vectorial escalable

#### **apple-touch-icon.png**
- **Tamaño**: 180x180 píxeles
- **Formato**: PNG con transparencia
- **Uso**: Icono para dispositivos iOS
- **Recomendación**: Logo sin fondo, esquinas redondeadas automáticas

## 🎨 **RECOMENDACIONES DE DISEÑO**

- **Colores**: Mantener consistencia con la paleta de la app
- **Estilo**: Minimalista, reconocible a pequeño tamaño
- **Fondo**: Transparente para adaptarse a diferentes temas
- **Texto**: Evitar texto pequeño, usar solo elementos gráficos

## 🔧 **HERRAMIENTAS PARA CREAR ICONOS**

- **Figma**: Diseño vectorial profesional
- **Canva**: Herramienta online gratuita
- **Inkscape**: Software gratuito de diseño vectorial
- **Real Favicon Generator**: https://realfavicongenerator.net/

## 📝 **INSTRUCCIONES DE INSTALACIÓN**

1. Diseñar el icono base (512x512 recomendado)
2. Exportar en los tamaños requeridos
3. Reemplazar los archivos placeholder actuales
4. Probar en diferentes dispositivos

## ⚠️ **NOTAS IMPORTANTES**

- Los iconos actuales son placeholders
- Deben ser archivos PNG válidos
- Verificar que se vean bien en diferentes tamaños
- Probar instalación de PWA en móviles
